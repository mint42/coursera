# Array Stats

This project performs simple stats operations on an array

Functions to implement:

- print_statistics()
- print_array()
- find_median()
- find_mean()
- find_maximum()
- find_minimum()
- sort_array()

## Getting Started

```
$> git clone https://github.com/mint42/coursera.git [name]
$> cd [name]
$> gcc stats.c
$> ./a.out
```
